## 1. 产品概述

### 1.1 产品定位

**数据结构学习助手** 是一款面向大学生的 Web 端数据结构可视化教学工具。通过交互式动画演示，将抽象的数据结构原理转化为直观的视觉呈现，帮助学习者理解核心数据结构的内部机制与操作过程。

### 1.2 核心价值主张

- **可视化理解** — 将抽象的数据结构转化为直观的图形动画
- **交互式学习** — 支持用户主动操作，观察数据结构的变化过程
- **即时反馈** — 每一步操作都有动画和日志反馈，强化学习效果
- **自主探索** — 支持随机生成、导入导出、分享，鼓励自主实验
- **引导式教学** — 学习模式提供步骤化引导，适合初学者循序渐进

### 1.3 目标用户

| 用户群体 | 特征 | 使用场景 |
|---------|------|---------|
| **大学生** | 计算机/软件工程专业在读 | 课程作业辅助、考试复习 |
| **自学者** | 正在学习数据结构的初学者 | 自学巩固、理解难点 |
| **教师** | 讲授数据结构课程的讲师 | 课堂演示、教学辅助 |

### 1.4 使用场景

1. **课堂学习** — 学生在课后使用工具回顾课堂内容，通过亲手操作加深理解
2. **作业辅助** — 遇到数据结构相关作业时，通过可视化验证自己的理解
3. **考前复习** — 通过反复操作和算法对比，巩固对排序算法等知识点的掌握
4. **教学演示** — 教师在课堂上投影展示数据结构操作过程
5. **算法对比实验** — 在 SortCompare 页面并行对比多种排序算法的执行过程

## 2. 功能需求

### 2.1 数据结构可视化（15 种）

| **二叉树 BinaryTree** | 插入、前序/中序/后序/层序遍历、查找、删除节点 | 树状层级布局 | 4 种遍历高亮动画 |
| **AVL 树 AVLTree** | 插入、删除、查找、自平衡旋转 | 树状层级布局 + 平衡因子 | 旋转动画与高度变化 |

| **排序 Sorting** | 12 种排序算法演示 | 柱状图高度表示值 | 暂停/继续、单步执行 |
| **算法对比 SortCompare** | 12 种排序算法并行对比 | 多组并行柱状图 + 性能图表 | 实时进度追踪 |

## 3. 详细功能规格

### 3.1 数组 Array

**数据模型:** `number[]`，初始值 `[8, 3, 12, 5, 9]`

**操作规格:**

| 操作 | 输入 | 输出 | 错误处理 |
|------|------|------|---------|
| 插入 | 数值 (1~99)，索引 (0~length) | 在指定位置插入元素 | 索引越界 Toast 错误；数组已满（≥20）拒绝插入 |
| 删除 | 索引 (0~length-1) | 删除指定位置元素 | 索引越界 Toast 错误 |
| 查找 | 数值 (1~99) | 返回索引（-1 表示未找到） | 未找到 Toast 提示 |
| 随机生成 | 无 | 生成 8 个 1~99 的随机数 | 无 |
| 重置 | 无 | 恢复初始值 | 无 |

**可视化规格:**
- 矩形条宽度 60px，高度 50px，间距 10px
- 元素值显示在矩形中央
- 索引标注显示在矩形下方
- 插入动画：高亮目标位置 + 指示器引导
- 删除动画：目标元素变红 + 下落淡出
- 搜索动画：顺序高亮对比，找到后绿色高亮

### 3.2 栈 Stack

**数据模型:** `number[]`，栈顶在数组末尾（右侧）

**操作规格:**

| 操作 | 输入 | 输出 | 错误处理 |
|------|------|------|---------|
| Push | 数值 (1~99) | 元素入栈 | 栈满（≥20）Toast 错误 |
| Pop | 无 | 栈顶元素出栈 | 栈空 Toast 错误 |
| Peek | 无 | 返回栈顶元素（不出栈） | 栈空 Toast 错误 |
| Clear | 无 | 清空栈 | 无 |

**可视化规格:**
- 垂直堆叠，栈顶在上
- 新元素 Push：从上方飞入
- 元素 Pop：向上飞出并淡出
- 栈底和栈顶标记

### 3.3 队列 Queue

**数据模型:** `number[]`，队首在数组开头（左侧）

**操作规格:**

| 操作 | 输入 | 输出 | 错误处理 |
|------|------|------|---------|
| Enqueue | 数值 (1~99) | 元素入队 | 队列满（≥20）Toast 错误 |
| Dequeue | 无 | 队首元素出队 | 队列空 Toast 错误 |
| Front | 无 | 返回队首元素（不出队） | 队列空 Toast 错误 |
| Clear | 无 | 清空队列 | 无 |

**可视化规格:**
- 水平排列，队首在左，队尾在右
- 新元素 Enqueue：从右侧滑入
- 元素 Dequeue：向左滑出并淡出

### 3.4 链表 LinkedList

**数据模型:** `{ value: number, next: Node | null }`，含头指针

**操作规格:**

| 操作 | 输入 | 输出 | 错误处理 |
|------|------|------|---------|
| 头插 | 数值 (1~99) | 在头部插入新节点 | 长度 ≥20 Toast 错误 |
| 尾插 | 数值 (1~99) | 在尾部插入新节点 | 长度 ≥20 Toast 错误 |
| 按位置插入 | 数值 (1~99)，位置 (0~length) | 在指定位置插入 | 位置越界 Toast 错误 |
| 删除 | 位置 (0~length-1) | 删除指定位置节点 | 位置越界 Toast 错误；空链表 Toast 错误 |
| 查找 | 数值 (1~99) | 返回位置（-1 表示未找到） | 未找到 Toast 提示 |
| 反转 | 无 | 反转链表方向 | 空链表 Toast 错误 |
| 环检测 | 无 | 检测是否存在环 | 空链表 Toast 错误 |

**可视化规格:**
- 圆形节点（半径 25px）+ 带箭头连线
- 节点值显示在圆内
- 反转动画：节点位置互换 + 箭头方向翻转
- 环检测：快慢指针动画，发现环后高亮环上节点

### 3.5 二叉树 BinaryTree

**数据模型:** `{ value: number, left: TreeNode | null, right: TreeNode | null }`

**操作规格:**

| 操作 | 输入 | 输出 | 错误处理 |
|------|------|------|---------|
| 插入 | 数值 (1~99) | 按 BST 规则插入 | 数值已存在 Toast 提示 |
| 删除节点 | 数值 (1~99) | 删除对应节点并调整树 | 节点不存在 Toast 错误 |
| 查找 | 数值 (1~99) | 高亮目标节点路径 | 未找到 Toast 提示 |
| 前序遍历 | 无 | 按前序顺序高亮节点 | 空树 Toast 提示 |
| 中序遍历 | 无 | 按中序顺序高亮节点 | 空树 Toast 提示 |
| 后序遍历 | 无 | 按后序顺序高亮节点 | 空树 Toast 提示 |
| 层序遍历 | 无 | 按层级顺序高亮节点 | 空树 Toast 提示 |

**可视化规格:**
- 树状层级布局，根节点在上
- 父子节点用线连接
- 节点插入：从根节点开始比较路径高亮，新节点生长动画
- 遍历：按顺序高亮节点，已遍历节点保持特殊颜色

### 3.6 图 Graph

**数据模型:** `nodes: { id, x?, y? }[], links: { source, target, weight? }[]`

**操作规格:**

| 操作 | 输入 | 输出 | 错误处理 |
|------|------|------|---------|
| 添加节点 | 无 | 生成新节点（ID: G-Z → N1, N2...） | 节点数 ≥26 Toast 错误 |
| 删除节点 | 节点 ID | 删除节点及关联边 | 节点不存在 Toast 错误 |
| 添加边 | 源节点 ID，目标节点 ID，权重 (1~99) | 创建有向边 | 自环/重复边 Toast 错误 |
| 删除边 | 源节点 ID，目标节点 ID | 删除边 | 边不存在 Toast 错误 |
| BFS | 起始节点 ID | 层级遍历顺序高亮 | 起始节点不存在 Toast 错误 |
| DFS | 起始节点 ID | 深度遍历顺序高亮 | 起始节点不存在 Toast 错误 |
| Dijkstra | 起始节点 ID，目标节点 ID | 最短路径高亮 + 距离 | 节点不存在 Toast 错误 |

**视图模式:**
- **力导向图** — 默认，节点间有物理斥力/引力模拟
- **邻接矩阵** — 矩阵形式展示节点连接关系
- **邻接表** — 列表形式展示每个节点的邻居

**可视化规格:**
- 力导向图：圆形节点 + 带箭头连线，连线标注权重
- 节点拖拽可调整位置
- 算法执行时按访问顺序高亮节点和边

### 3.7 排序算法（8 种）

**数据模型:** `number[]`，初始随机生成 10 个数值

**支持算法:**

| 算法 | 时间复杂度 | 空间复杂度 | 稳定性 |
|------|-----------|-----------|--------|
| 冒泡排序 | O(n²) | O(1) | 稳定 |
| 选择排序 | O(n²) | O(1) | 不稳定 |
| 插入排序 | O(n²) | O(1) | 稳定 |
| 快速排序 | O(n log n) | O(log n) | 不稳定 |
| 归并排序 | O(n log n) | O(n) | 稳定 |
| 堆排序 | O(n log n) | O(1) | 不稳定 |
| 基数排序 | O(d×(n+k)) | O(n+k) | 稳定 |
| 桶排序 | O(n+k) | O(n+k) | 稳定 |

**操作规格:**
- 选择算法后自动执行并播放动画
- 支持暂停/继续（Space 键）
- 显示比较次数、交换次数、总步数统计
- 排序完成后所有元素绿色高亮

**可视化规格:**
- 柱状图，高度表示数值大小
- 比较时两柱高亮（黄色）
- 交换时两柱位置互换动画
- 排序完成时全部变绿

### 3.8 哈希表 Hash Table

**数据模型:** 桶数组 `buckets: (number | null)[][]`，哈希函数 `hash(key) = key % bucketCount`

**操作规格:**

| 操作 | 输入 | 输出 | 错误处理 |
|------|------|------|---------|
| 插入 | 数值 (1~99) | 计算哈希值，插入对应桶 | 数值已存在 Toast 提示；表满 Toast 错误 |
| 删除 | 数值 (1~99) | 查找并删除 | 数值不存在 Toast 错误 |
| 查找 | 数值 (1~99) | 高亮查找路径 | 数值不存在 Toast 提示 |

**可视化规格:**
- 桶数组水平排列
- 每个桶内冲突元素垂直堆叠
- 冲突元素用箭头连接到桶
- 插入时高亮目标桶

### 3.9 堆 Heap

**数据模型:** 最大堆，`number[]`，满足父节点 ≥ 子节点

**操作规格:**

| 操作 | 输入 | 输出 | 错误处理 |
|------|------|------|---------|
| Insert | 数值 (1~99) | 插入并上浮调整 | 堆满（≥31）Toast 错误 |
| ExtractMax | 无 | 移除根节点并下沉调整 | 堆空 Toast 错误 |
| Peek | 无 | 返回根节点（不移除） | 堆空 Toast 错误 |

**可视化规格:**
- 完全二叉树层级布局
- 节点值显示在圆内
- Insert：新节点在底部生长，上浮路径高亮
- ExtractMax：根节点移除，底部节点移到根部，下沉路径高亮
- 违规检测：子节点 > 父节点时红色高亮

### 3.10 字典树 Trie

**数据模型:** `{ children: Map<string, TrieNode>, isEnd: boolean }`

**操作规格:**

| 操作 | 输入 | 输出 | 错误处理 |
|------|------|------|---------|
| 插入 | 字符串（小写字母 a-z） | 逐字符插入，必要时创建新节点 | 长度 >20 Toast 错误；非法字符 Toast 错误 |
| 删除 | 字符串 | 删除单词，清理无用节点 | 单词不存在 Toast 错误 |
| 查找 | 字符串 | 高亮查找路径 | 单词不存在 Toast 提示 |
| 前缀匹配 | 前缀字符串 | 返回所有匹配单词列表 | 无匹配 Toast 提示 |

**可视化规格:**
- 树形层级布局
- 每条边标注对应字符
- `isEnd=true` 的节点特殊标记（双重圆圈）
- 查找时路径高亮

### 3.11 排序算法对比 SortCompare

**功能规格:**
- 同时运行 8 种排序算法
- 每组使用相同初始随机数据
- 实时显示每组排序进度
- 排序完成后显示 PerformanceChart（比较次数/交换次数/总步数）
- 支持 Timeline 查看各算法执行历史

### 3.12 图算法页面 GraphAlgorithmPage

**功能规格:**
- 选择算法类型（BFS/DFS/Dijkstra/拓扑排序）
- 提供预设图或自定义图
